# Road Network Pattern

This ontology specifies the service-level concepts for road transport in the city data model.

This pattern imports the following patterns:

- [Transport Network Pattern](TransportNetworkPattern.md)

This pattern consists of the following classes:

- [Road](Road.md)
- [Road Lane](RoadLane.md)
- [Road Link](RoadLink.md)
- [Road Link User Type](RoadLinkUserType.md)
- [Road Network](RoadNetwork.md)
- [Road Section](RoadSection.md)
- [Road Segment](RoadSegment.md)
- [Vehicle Flow Rate](VehicleFlowRate.md)

The formal definition of this pattern is available in [OWL Syntax](RoadNetworkPattern.owl).
