# Rail Network Pattern

This ontology specifies the service-level concepts for rail transport in the city data model.

This pattern imports the following patterns:

- [Transport Network Pattern](TransportNetworkPattern.md)

This pattern consists of the following classes:

- [Rail Corridor](RailCorridor.md)
- [Rail Network](RailNetwork.md)
- [Rail Section](RailSection.md)
- [Track Link](TrackLink.md)
- [Track Segment](TrackSegment.md)

The formal definition of this pattern is available in [OWL Syntax](RailNetworkPattern.owl).
