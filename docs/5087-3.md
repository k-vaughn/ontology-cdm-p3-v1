# 5087-3

This ontology specifies the service-level concepts defined for the transport service of the city data model.

This pattern imports the following patterns:

- [Micromobility Network Pattern](MicromobilityNetworkPattern.md)
- [Pedestrian Network Pattern](PedestrianNetworkPattern.md)
- [Public Transport System Pattern](PublicTransportSystemPattern.md)
- [Rail Network Pattern](RailNetworkPattern.md)
- [Road Network Pattern](RoadNetworkPattern.md)
- [Transport Alert Pattern](TransportAlertPattern.md)
- [Travel Corridor Pattern](TravelCorridorPattern.md)

This pattern does not contain any classes.


The formal definition of this pattern is available in [OWL Syntax](5087-3.owl).
