# Pedestrian Network Pattern

This ontology specifies the pedestrian network pattern of the transport service in the city data model.

This pattern imports the following patterns:

- [Transport Network Pattern](TransportNetworkPattern.md)

This pattern consists of the following classes:

- [Footpath](Footpath.md)
- [Footpath Lane](FootpathLane.md)
- [Footpath Link](FootpathLink.md)
- [Footpath Network](FootpathNetwork.md)
- [Footpath Section](FootpathSection.md)
- [Footpath Segment](FootpathSegment.md)

The formal definition of this pattern is available in [OWL Syntax](PedestrianNetworkPattern.owl).
