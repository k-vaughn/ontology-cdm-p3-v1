# Micromobility Network Pattern

This ontology specifies the micromobility network pattern of the transport service of the city data model.

This pattern imports the following patterns:

- [Transport Network Pattern](TransportNetworkPattern.md)

This pattern consists of the following classes:

- [Micromobility Lane](MicromobilityLane.md)
- [Micromobility Link](MicromobilityLink.md)
- [Micromobility Network](MicromobilityNetwork.md)
- [Micromobility Path](MicromobilityPath.md)
- [Micromobility Path Section](MicromobilityPathSection.md)
- [Micromobility Path Segment](MicromobilityPathSegment.md)

The formal definition of this pattern is available in [OWL Syntax](MicromobilityNetworkPattern.owl).
