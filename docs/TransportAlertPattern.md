# Transport Alert Pattern

This ontology specifies the city-level concepts for the transport alert pattern for the transport domain.

This pattern imports the following patterns:

- Alert Pattern
- [Transport Network Pattern](TransportNetworkPattern.md)

This pattern consists of the following classes:

- [Transport Alert](TransportAlert.md)
- [Transport Alert Thing](TransportAlertThing.md)

The formal definition of this pattern is available in [OWL Syntax](TransportAlertPattern.owl).
