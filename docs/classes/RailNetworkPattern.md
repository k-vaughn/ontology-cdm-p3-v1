# Rail Network Pattern

The rail network pattern specializes the transport network pattern to address the needs for travel of rail vehicles (e.g., trains, subways, trams).

It consists of the following classes:

- [Rail Corridor](RailCorridor.md)
- [Rail Network](RailNetwork.md)
- [Rail Section](RailSection.md)
- [Track Link](TrackLink.md)
- [Track Segment](TrackSegment.md)
